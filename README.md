# HighApi
The easiest way to create a simple api for testing and prototyping with .NET using top level programming feature in C# 9 and the power of abstraction.

## Goal
* Make it easy to recommend C# and .NET for building RESTful APIs
* Create a temp API for testing and prototyping in less than 12 lines of code
* Focus on shaping the data from the start

## Get started
1) Create a new console app
2) Open your .csproj file and enable preview flag like this :
     
       <PropertyGroup>
          <TargetFramework>net5.0</TargetFramework>
          <LangVersion>preview</LangVersion>
       </PropertyGroup>

4) Add reference to the [nuget package](https://www.nuget.org/packages/HighApi/)
5) Start your journey
